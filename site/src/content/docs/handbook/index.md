Welcome to the Backpropagate handbook. This is the complete operator guide — every flag, every env var, every error code, every recipe — from first install through training, export, and running your fine-tune locally with Ollama. Start at [Getting Started](/backpropagate/handbook/getting-started/) if you've just installed; jump to [Troubleshooting](/backpropagate/handbook/troubleshooting/) if something is on fire; head to [Recipes](/backpropagate/handbook/recipes/) for paste-and-run snippets matched to common operator workflows.
